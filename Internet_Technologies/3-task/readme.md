# Калькулятор IP-адресов

Этот проект вычисляет диапазон IP-адресов, номер узла и номер сети на основе введённого IP-адреса и маски подсети. Проект поддерживает два режима: с наложением подсетей (когда маска задаётся вручную) и без наложения (когда используется стандартная маска для класса сети).

## Как использовать
1. Выберите режим работы:
   - **Без наложения подсетей**: маска определяется автоматически на основе класса сети IP-адреса (A, B или C).  
   - **С наложением подсетей**: вы сами задаёте маску в формате `/xx` (например, `/24`) или `xxx.xxx.xxx.xxx` (например, `255.255.255.0`).
2. Введите IP-адрес (например, `192.168.5.100`).
3. Если выбран режим "с наложением", введите маску.
4. Выберите задачу:  
   - Диапазон IP-адресов.  
   - Номер узла.  
   - Номер сети.  
5. Нажмите "Выполнить" и посмотрите результат.

## Файлы
- `index.html` — главная страница для режима "без наложения".  
- `overlay.html` — страница для режима "с наложением".  
- `index.css` — стили для `index.html`.  
- `overlay.css` — стили для `overlay.html`.  
- `index.js` — логика для режима "без наложения".  
- `overlay.js` — логика для режима "с наложением".
