# Многопоточное суммирование

Программа суммирует 1 миллион случайных чисел с помощью 4 потоков.

## Что делает программа

1. Создаёт массив из 1 000 000 элементов
2. Заполняет его случайными числами от 0 до 99
3. Делит массив на 4 части
4. Каждый поток считает сумму своей части параллельно
5. Складывает результаты

## Компиляция

```bash
g++ -std=c++11 -pthread main.cpp -o program
./program
```

## Пример вывода

```
время заполнения: 0.015792 сек
сумма: 49458832
время: 0.006134 сек
```

## Как работает

Массив разбивается на 4 части:
- Поток 1: элементы 0-249999
- Поток 2: элементы 250000-499999
- Поток 3: элементы 500000-749999
- Поток 4: элементы 750000-999999

Все потоки работают одновременно, поэтому программа быстрее однопоточной версии.
