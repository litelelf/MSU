# Многопоточное суммирование

Программа суммирует 1 миллион случайных чисел двумя способами: с помощью 4 потоков и без потоков. Сравнивает время выполнения.

## Что делает программа

1. Создаёт массив из 1 000 000 элементов
2. Заполняет его случайными числами от 0 до 99
3. Считает сумму **с 4 потоками** (параллельно)
4. Считает сумму **без потоков** (последовательно)
5. Сравнивает время выполнения

## Компиляция

```bash
g++ -std=c++11 -pthread main.cpp -o program
./program
```

## Пример вывода

```
время заполнения: 0.013505 сек
сумма без потоков: 49485834
время без потоков: 0.008328 сек
сумма с потоками: 49485834
время с потоками: 0.006402 сек
```

## Как работает

### С потоками
Массив разбивается на 4 части:
- Поток 1: элементы 0-249999
- Поток 2: элементы 250000-499999
- Поток 3: элементы 500000-749999
- Поток 4: элементы 750000-999999

Все потоки работают одновременно.

### Без потоков
Один поток последовательно проходит по всему массиву.

## Функции

- `fillVector()` - заполняет массив случайными числами
- `sumPart()` - считает сумму части массива (для потоков)
- `SumAll()` - считает сумму всего массива (без потоков)
